## Conventions  

### Versioning  
[Semantic Versioning](https://semver.org/) is used.  

### Commits  
Each commit to the main branch follows the conventions of [Conventional Commits](https://www.conventionalcommits.org/).  

### Folder structure  
* src - source code  
* doc - documentation  
* cmd - auxiliary scripts
* asm - assemblies (generated)
